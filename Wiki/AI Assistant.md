# ИИ-Ассистент (Генерация вакансий)

В бота встроен мощный модуль ИИ-генерации текстов для вакансий. Он использует провайдера `OpenRouter` для доступа к современным LLM (Large Language Models).

## Как это работает?
Вместо того, чтобы писать пост вручную, вы можете нажать **🤖 Сгенерировать пост**.
Бот предложит выбрать тип вакансии (например: `Удаленная работа` или `Курьер / Доставщик`).

Затем бот:
1. Запрашивает у OpenRouter текст вакансии на основе заранее заданного системного промпта.
2. Подбирает случайную ссылку из вашего конфига `vacancy_config.json`.
3. Автоматически прикрепляет нужную картинку (например `remote.jpg` или `courier.jpg`).
4. Автоматически создает кнопки: "Откликнуться" (с выбранной ссылкой) и "Написать HR-менеджеру" (постоянная ссылка на HR).
5. Присылает вам готовый пост на проверку.

## Управление конфигами вакансий
Вся логика для вакансий хранится в файле `vacancy_config.json`.
```json
{
  "remote": {
    "system_prompt": "Ты - HR эксперт. Напиши огненную вакансию...",
    "image": "photos/remote.jpg",
    "links": [
      {"url": "https://ссылка1", "weight": 1},
      {"url": "https://ссылка2", "weight": 1}
    ]
  }
}
```
Здесь вы можете гибко менять промпты (инструкции) для ИИ и добавлять новые реферальные ссылки для кнопок.

## Быстрая генерация
Кнопка `⚡ Быстрая генерация` случайным образом выбирает тип вакансии (из настроенных в конфиге) и генерирует пост в один клик. Идеально подходит для массовой заготовки постов!
