# Настройка бота (Configuration)

Бот настраивается через файл `.env`.

## Переменные окружения (.env)
Создайте файл `.env` в корневой папке проекта со следующим содержимым:
```env
BOT_TOKEN=8908175250:ваш_токен_от_BotFather
ADMIN_ID=123456789
TARGET_CHANNEL=@ваш_канал
OPENROUTER_KEY=sk-or-v1-...
```

- **BOT_TOKEN**: Токен вашего Telegram-бота.
- **ADMIN_ID**: Ваш цифровой ID в Telegram. Только этот пользователь сможет взаимодействовать с ботом.
- **TARGET_CHANNEL**: Юзернейм (например `@cadrypro_official`) или ID канала, куда будут публиковаться посты. Бот должен быть администратором в этом канале с правом отправки сообщений.
- **OPENROUTER_KEY**: Ваш API-ключ от платформы OpenRouter. Без него меню генерации ИИ-вакансий не будет работать.

## Зависимости
Для работы бота необходим Python 3.10 или новее.
Пакеты для установки (через `pip`):
- `python-telegram-bot` (версия 20.x+)
- `python-dotenv`
- `httpx` (используется для асинхронных запросов к OpenRouter)

Установка: `pip install -r requirements.txt` (если есть) или вручную.

## Данные
Все созданные посты сохраняются локально в файле `posts.json`. Если бот перезапустится, он прочитает этот файл и продолжит работу, а фоновые задачи (расписание) автоматически восстановятся.
