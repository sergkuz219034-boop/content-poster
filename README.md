# Content Manager Bot 🤖📱

**Content Manager Bot** — это мощный Telegram-бот для управления контентом и автоматической публикации постов в каналах. Он позволяет администраторам создавать посты с медиавложениями, добавлять интерактивные кнопки, планировать отложенные публикации и даже генерировать готовые вакансии с помощью нейросетей.

## Основные возможности 🌟

*   **📝 Создание постов:** Удобный конструктор для текста с поддержкой HTML-форматирования (жирный, курсив, подчеркивание).
*   **🖼 Медиавложения:** Поддержка прикрепления фотографий и видео к постам.
*   **🔗 Интерактивные кнопки:** Легкое добавление inline-кнопок со ссылками. Поддержка цветовых тегов (`!r`, `!g`, `!b`) для выделения кнопок (зависит от клиента Telegram).
*   **📅 Отложенная публикация:** Возможность опубликовать пост "прямо сейчас" или запланировать на любую дату и время в будущем с помощью удобного календаря/часов.
*   **🤖 ИИ-генерация вакансий:** Интеграция с OpenRouter для автоматического создания привлекательных текстов вакансий (например, удаленка или курьер) по заданным шаблонам.
*   **📋 Управление постами:** Просмотр списка всех запланированных и опубликованных постов. Возможность редактирования текста, изменения даты публикации, дублирования и удаления.
*   **⚙️ Настройки и статистика:** Просмотр текущих настроек, ID администратора, канала для публикации и общей статистики бота.

## Требования 🛠

*   Python 3.10+
*   Библиотека `python-telegram-bot` v20+
*   Библиотеки `python-dotenv`, `httpx`

## Установка и запуск 🚀

1.  Клонируйте репозиторий.
2.  Создайте виртуальное окружение и установите зависимости:
    ```bash
    pip install python-telegram-bot python-dotenv httpx
    ```
3.  Создайте файл `.env` в корне проекта и заполните его:
    ```env
    BOT_TOKEN=ваш_токен_бота_от_BotFather
    ADMIN_ID=ваш_telegram_id
    TARGET_CHANNEL=@ваш_канал
    OPENROUTER_KEY=ваш_ключ_от_openrouter # (опционально, для ИИ)
    ```
4.  Запустите бота:
    ```bash
    python bot.py
    ```
